🚀 CÓMO EJECUTAR
════════════════════════════════════════════════════════════════════

1. VERIFICAR QUE TODO FUNCIONA:
   python check.py
   
2. INSTALAR DEPENDENCIAS (si no las tiene):
   pip install -r requirements-simple.txt
   
3. INICIAR EL SERVIDOR:
   python main.py
   
   El servidor se ejecutará en: http://localhost:8000
   
4. ACCEDER A LA DOCUMENTACIÓN:
   http://localhost:8000/docs
   
5. EJECUTAR PRUEBAS:
   python test_api.py

🎯 CARACTERÍSTICAS IMPLEMENTADAS
════════════════════════════════════════════════════════════════════

✅ Análisis de Emociones
   - 5 emociones primarias (Enojo, Alivio, Confusión, Ansiedad, Neutral)
   - Emociones secundarias
   - Confianza porcentual

✅ Procesamiento de Lenguaje Natural (NLP)
   - Extracción de temas principales
   - Identificación de palabras clave
   - Análisis de enunciado del problema

✅ Cálculo de Indicadores
   - Nivel de interés del cliente (Bajo, Medio, Alto)
   - Nivel de estrés (Normal, Moderado, Crítico)
   - Urgencia (Baja, Media, Inmediata)
   - Satisfacción (porcentaje)

✅ Recomendaciones Inteligentes
   - Basadas en emoción y urgencia
   - Acciones específicas para el agente
   - Prioridades claras

✅ API RESTful
   - Documentación automática (Swagger/ReDoc)
   - Validación de datos con Pydantic
   - Manejo de errores
   - CORS configurado

📚 SERVICIOS IMPLEMENTADOS
════════════════════════════════════════════════════════════════════

1. EmotionAnalysisService
   Detecta emociones usando análisis de palabras clave
   Retorna: emoción primaria, confianza, emociones secundarias
   
2. NLPService  
   Extrae temas, keywords, y enunciados de problemas
   Utiliza expresiones regulares y análisis de palabras
   
3. TranscriptionService
   Placeholder para integración con Whisper/Google Speech-to-Text
   En desarrollo: soporte para audio en tiempo real
